# Calculadora de Tarifas de Autopistas 

Programa sencillo en Python para calcular la tarifa según el tipo de vehículo.

---

##  Descripción

Esta calculadora permite determinar la tarifa de peaje en autopistas según el tipo de vehículo ingresado (`auto`, `moto`, `camion`, `bus`).  
Es ideal para practicar lógica, entrada de datos por consola y estructuras condicionales básicas.

---

##  Cómo usar

1. Clona o descarga este repositorio.  
2. Asegúrate de tener Python instalado (idealmente Python 3).  
3. En consola, ve al directorio del proyecto y ejecuta:

python calculadora_autopistas.py

4. Cuando te pida “Ingrese tipo de vehículo ...”, escribe uno de los siguientes:  
   - `auto`  
   - `moto`  
   - `camion`  
   - `bus`  

5. El programa mostrará el valor de la tarifa correspondiente.
